## Patakazi jobsite

12/04/2024

Patakazi is one page jobsite that helps job seekers fast. The code uses https://jsonfakery.com/jobs to populate the jobs.

### Features

- [x] Search jobs by name
- [x] Random 10 new job on page reload

### Technology

- [x] HTML
- [x] CSS
- [x] Jacript
